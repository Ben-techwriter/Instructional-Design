
# Instructional Design Sample Project

This repository showcases an end‑to‑end Instructional Design sample, including storyboards, learning objectives, and design rationale. 

This project is designed to demonstrate the complete ID workflow:
- Analysis
- Design
- Development
- Revision
- Delivery

## What This Repository Contains
- Learning Objectives aligned to Blooms Taxonomy
- Storyboard showcasing audio visuals and interactions
- Design rationale showcasing the choice of design and strategy
